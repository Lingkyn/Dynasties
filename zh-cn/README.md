# Dynasties

> 一款以唐、宋、元、明四大朝代为背景的综合策略桌游。

在 **Dynasties** 中，玩家将分别扮演中国古代的四个朝代阵营（唐 / 宋 / 元 / 明），通过军事、民生、财富与信仰的多系统治理，在扩张、发展、征税、灾祸应对与政策制定之间不断权衡，争夺最后的存活与统治权。

## 游戏概览

- **四大阵营**：  
  唐 / 宋 / 元 / 明，各自拥有不同的初始属性与官职、政策组合，代表各自的制度特点与发展方向。  

- **四维属性系统**：  
  - **军事值**：决定战力，并参与入侵、叛乱、政变等事件判定。  
  - **民生值**：决定每轮初始获得的子民数量，过低时叛乱事件将会生效。  
  - **财富值**：决定每轮初始获得的钱币数量，过高时腐败事件将会生效。  
  - **信仰值**：阵营的"生命值"；其他属性降为 0 时会消耗信仰。  

- **治理转盘**：  
  每回合通过转盘进入不同系统： **军事 / 民生 / 财富 / 信仰**，分别处理扩张、民生、税收和灾祸应对等事务。  

- **天灾人祸系统**：  
  干旱、地震、洪水、瘟疫等天灾，以及叛乱、入侵、政变、腐败等人祸，会在特定条件下触发，考验玩家的资源调配和风险管理。  

- **官职与政策系统**：  
  通过不同的官职执行 / 颁布政策牌，玩家可以对灾祸做出应对、调整属性，形成各阵营独特的治理路线。  

---

## 胜利条件

- 当任意玩家的 **信仰值** 归 0 时，该玩家立即 **淘汰**；  
- 当场上只剩下 **最后一名存活玩家** 时，该玩家获得胜利。
