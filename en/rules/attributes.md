# Attribute System

## Military

* Battle Power = Faction Military + number of citizen tokens
* Participates in judgments for invasion, rebellion, and coup events

## Livelihood

* Determines the number of citizen tokens gained each turn
* When Livelihood ≤ 3, rebellion events are triggered

## Wealth

* Determines coins gained each turn: Wealth × 500
* When Wealth ≥ 4, corruption events are triggered

## Faith
> Faction's life points

* If Military / Livelihood / Wealth drops to 0, Faith -1, and the attribute resets to 1
